## Chức năng

Hỗ trợ Steam, Nox, SandBox. Tính năng ngự hồn, tham sân si, Himeko, Phá kết giới, Thám hiểm ...

## Môi trường làm việc

Onmyoji steam độ phân giải mặc định. Độ phân giải giả lập Nox (1136x640); Hệ thống win 10, màn hình (1920x1080).

Công cụ có thể chạy nền nhưng **Không được thu nhỏ**.
